# Rapidly exploring Random Tree algorithms

# Overview 

This repository implements some common RRT algorithms used in robotics ( in our case we used car like mobile robot case).

## Prerequisites 
  python 3.10

### Instalation
`pip install -r requirements.txt`

## Test
##### RRT:   
`py test.py -t rrt`
